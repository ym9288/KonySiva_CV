TeddiBot Privacy Agreement

The TeddiBot extension interacts with the TeddiBot Desktop App during use, sending messages to the extension via the app, notifying the extension to fetch page elements, tags, URLs, etc. and send them to the app.

The App uses this information to perform data entry on the page, button clicks, tab switching and other operations to automate web operations.

TeddiBot does not store or publish this information. It only retains information about what the user has done on the page and then restores these actions to automate them.

If you are a TeddiBot user, please ignore and close the page.

If you are not using the TeddiBot software or do not wish to have your browser information captured, please click the below 'uninstall' button to remove the extension.
